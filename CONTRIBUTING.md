# Contributing

Thank you for contributing to this curated paper list.

## Add a paper

1. Add one row to `data/papers.csv`.
2. Use one of the supported task labels: `DTI/DTA`, `DRP`, `DPP`, `MPP`, `DDI`, `MG`, `MO`.
3. Prefer official links for `paper` and `code`.
4. Add a short `notes` sentence explaining the contribution.
5. Keep method names and venue names consistent with the original paper.

## Suggested tags

- Representations: `SMILES`, `2D graph`, `3D graph`, `protein sequence`, `binding pocket`, `omics`, `KG`, `electronic density`.
- Architectures: `GNN`, `MPNN`, `GAT`, `Graph Transformer`, `PLM`, `Diffusion`, `VAE`, `GAN`, `Flow`, `GFlowNet`.
- Evaluation: `IID`, `Unseen-drug`, `Unseen-target`, `Unseen-cell`, `Unseen-both`, `OOD`.

## Pull request checklist

- [ ] The paper is assigned to the correct task.
- [ ] The paper title/method name is accurate.
- [ ] Paper and code links are official when possible.
- [ ] The dataset/evaluation setting is specified.
- [ ] BibTeX key is unique.
